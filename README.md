# KC: Divy světa
Vytvořte webovou stránku o **jednom ze sedmi divů světa**. Složka s projektem se bude jmenovat `7divu` a bude **publikovaná na webovém serveru TuX** na adrese (URL): 

```
https://tux.panska.cz/~10XPrijmeniJ/7divu 
```

## HTML

- Dokument HTML bude mít název `index.html` s úplnou HTML hlavičkou.
- V titulku bude název zvoleného divu.
- Dále postupujte podle náhledu.

## CSS
1. Soubor se styly bude externí.
2. Písmo stránky bude bezpatkové.
3. Barva všech nadpisů bude jiná než výchozí černá.
4. Hlavní nadpis a odstavec se jménem bude vycentrovaný.
5. Hypertextový odkaz po najetí myší změní vzhled.
6. V odrážkovém seznamu divů bude ten popisovaný červeně.
7. Obsah na stránce bude mít šířku 600 px a svisle vycentrovaný.

Můžete přidat další styly dle vlastní úvahy, ale nepřehánět! Bude to bonus, ale oceněný pouze pochvalou, ne lepší známkou.
